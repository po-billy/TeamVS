# ⚔️ TeamVS

> **팀을 나누고, 겨루고, 증명하라.**

내전·스크림의 팀짜기부터 게임 전적조회·전투력 랭킹, 뽑기(갓챠)·경제, 서버관리까지 갖춘 수익화 지향 디스코드 봇입니다.

**기술 스택**: Python 3.13 · discord.py 2.5+ · SQLite(aiosqlite)
**지원 게임**: 리그 오브 레전드 · 배틀그라운드 · 오버워치 · 발로란트 (확장 가능한 구조)

## 🚀 설치 & 실행

### 1. 디스코드 봇 만들기
1. https://discord.com/developers/applications → **New Application** (이름: TeamVS)
2. **Bot** 탭 → **Reset Token** → 토큰 복사
3. 같은 화면에서 **Privileged Gateway Intents** 중 **SERVER MEMBERS INTENT를 반드시 켜기**
4. **Installation** 탭에서 `bot` + `applications.commands` 스코프로 초대 링크 생성 후 서버에 초대

### 2. 게임 API 키 준비 (쓸 게임만)

| 게임 | 발급처 | 비고 |
|---|---|---|
| 롤 | https://developer.riotgames.com | 개발 키는 **24시간마다 갱신**. 정식 운영 전 Production Key 심사 필요 |
| 배그 | https://developer.pubg.com | 무료, 10요청/분 |
| 발로란트 | https://api.henrikdev.xyz/dashboard | HenrikDev 디스코드 가입 후 무료 Basic 키 (Riot 공식 API는 발로란트 별도 승인이 필요해 커뮤니티 표준 API 사용) |
| 오버워치 | 불필요 | OverFast 공개 API (유저의 배틀넷 프로필이 '공개'여야 조회 가능) |

키를 안 넣은 게임 기능은 자동으로 "준비 중" 안내를 띄우고, 나머지 기능은 정상 작동합니다.

### 3. 설정 & 실행
```powershell
copy .env.example .env
notepad .env        # DISCORD_TOKEN + 게임 API 키 입력

python -m venv .venv        # .venv가 이미 있으면 생략
.venv\Scripts\pip install -r requirements.txt
.venv\Scripts\python bot.py
```
또는 `start.bat` 더블클릭.

> 💡 명령어는 전역으로 등록되며 바로 반영됩니다. 만약 명령어가 **두 개씩** 보인다면
> (과거 서버 전용 등록과 중복) `.env`의 `TEST_GUILD_ID`에 그 서버 ID를 넣고 재시작하면 자동 정리됩니다.

### 4. koreanbots 등록 (선택 — 홍보 & 투표 보상)
1. https://koreanbots.dev 에서 봇 등록 신청 (심사 있음)
2. 발급받은 토큰을 `.env`의 `KOREANBOTS_TOKEN`에 입력
3. 자동 활성화: 서버 수 30분마다 업데이트 + `/투표보상`(하트 투표 → 코인 → 하트 랭킹 상승 선순환)

## 💰 수익화 가이드 (중요)

### 지켜야 할 정책
- **Riot API 정책**: API로 얻은 데이터(전적 등)를 **유료 기능 뒤에 잠그면 안 됩니다.** 전적조회는 무료로 열고 다른 기능으로 수익화하세요. Production Key 신청 시 수익 모델 명시 필요.
- **디스코드 정책**: 실제 돈이 오가는 도박·현금 환전 금지. 뽑기/미니게임은 **현금 가치 없는 가상 코인만** 사용하도록 설계되어 있습니다.
- 유료 결제를 붙일 경우 **확률 공시** 권장 (뽑기 확률: `cogs/gacha.py`의 `RARITIES` — 일반 50 / 고급 25 / 희귀 15 / 영웅 7 / 전설 2.5 / 신화 0.5%).

### 추천 수익 모델 (단계별)
1. **성장기**: `/투표보상`으로 koreanbots 하트 확보 → 리스트 상위 노출 → 서버 수가 곧 자산
2. **후원**: 토스아이디/카카오페이 링크 + 후원자 전용 코인 보너스·칭호
3. **서버 프리미엄 구독**: 전투력 자동 갱신, 경매 커스텀 예산, 전용 뽑기 아이템 등
4. **광고/제휴**: 임베드 푸터 스폰서 문구, 게임 커뮤니티 제휴

## 📁 프로젝트 구조
```
bot.py                  # 엔트리포인트 (명령어 동기화, 에러 핸들러, koreanbots 루프)
config.py               # .env 로드 + 경제/갓챠 밸런스 상수
utils/
  database.py           # SQLite (유저·코인·게임계정·인벤토리·천장·서버설정)
  riot.py               # Riot API 클라이언트 + Data Dragon 챔피언 한글명
  power.py              # 롤 전투력 계산 공식
  gameapi.py            # 게임 전적 API 공용 헬퍼
  koreanbots_api.py     # koreanbots v2 (투표 확인, 서버 수 업데이트)
cogs/
  general.py            # 도움말, 핑, 초대링크, 봇정보
  lol.py                # /롤 그룹 (Riot API)
  pubg.py               # /배그 그룹 (PUBG 공식 API)
  overwatch.py          # /오버워치 그룹 (OverFast API)
  valorant.py           # /발로란트 그룹 (HenrikDev API)
  team.py               # /팀짜기 (랜덤/턴제 드래프트/경매)
  economy.py            # 출석/지갑/송금/코인랭킹/투표보상
  gacha.py              # 뽑기/도감/판매/미니게임
  admin.py              # 서버관리 + 환영 메시지
data/bot.db             # SQLite DB (자동 생성)
```

## 🎮 새 게임 추가하는 법
1. `cogs/` 에 새 파일 생성 (기존 게임 코그 중 비슷한 것을 복사 — 키 필요하면 `pubg.py`, 불필요하면 `overwatch.py` 참고)
2. `GAME = "게임식별자"` 만 바꾸면 DB(`game_accounts`)는 그대로 재사용됩니다
3. 그 게임의 전적 API에 맞게 `등록`/`전적`(전투력 계산)/`랭킹` 을 수정
4. `bot.py`의 `EXTENSIONS` 에 추가 → 재시작

## ⚖️ 밸런스 조정
`config.py` 상수: 출석/투표 보상, 뽑기 비용, 천장, 경매 예산.
전투력 공식: 롤 `utils/power.py`, 배그/옵치/발로 각 코그 상단의 `compute_*_power`.

## ❓ 자주 발생하는 문제
- **명령어가 두 개씩 보여요** → 전역 등록과 서버 전용 등록이 중복된 상태. `TEST_GUILD_ID`에 서버 ID를 넣고 재시작하면 정리됨 (이후 디스코드 Ctrl+R 새로고침)
- **명령어가 안 보여요** → 봇 초대 시 `applications.commands` 스코프 포함 여부 확인 후 디스코드 Ctrl+R
- **롤 전적이 안 돼요** → Riot 개발 키는 24시간마다 만료. 새 키로 갱신 후 재시작
- **오버워치 전적이 안 나와요** → 게임 내 [옵션 → 소셜]에서 경력 프로필 공개 설정
- **배그 닉네임을 못 찾아요** → 대소문자를 정확히 입력했는지, 플랫폼(스팀/카카오) 확인
- **`PrivilegedIntentsRequired` 오류** → 개발자 포털에서 SERVER MEMBERS INTENT 켜기
